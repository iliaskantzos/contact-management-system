 Contact Management System (Java)

Console εφαρμογή για διαχείριση επαφών.

 Περιγραφή

Το project επιτρέπει τη δημιουργία και διαχείριση επαφών μέσω console εφαρμογής.

Λειτουργίες

* Προσθήκη επαφής
* Επεξεργασία επαφής
* Διαγραφή επαφής
* Έλεγχος εγκυρότητας δεδομένων (validation)

 Δομή Project

* ConsoleApp.java → κύρια εφαρμογή (main)
* Contact.java → μοντέλο επαφής
* ContactManager.java → διαχείριση επαφών
* Validation.java → έλεγχοι δεδομένων

 Τεχνολογίες

* Java
* Object-Oriented Programming (OOP)

Πώς να τρέξει

1. Άνοιξε το project σε IntelliJ ή Eclipse
2. Εκτέλεσε το `ConsoleApp.java`

Σκοπός

Το project δημιουργήθηκε για εκπαιδευτικούς σκοπούς και εξάσκηση στη Java.
